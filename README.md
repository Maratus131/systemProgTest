# Задания после прохождения теста

## Задание 1

- Логика парсинга юрл находится в parser.py
- В файле urls.py находятся константы, то есть ссылки на сайты для парсинга и пути к файлам для записи результатов парсинга.
- В файле utils.py находится функция для записи текста в файл с форматом csv.
- В зависимости от способа решения есть асинхронное (task_1_async.py) и многопоточное (task_1_threads.py) решение.
- Запуск происходит на клиенте.
- Итоговые цены для разных методов иногда могут отличаться, так как на сайтах имеются динамически изменяющиеся товары. 


## Задание 2

- Сгенерировать данные можно в generation_mocks_data.py
- В main.py находится сама логика чтения файлов и ее многопоточная и асинхронная реализация
